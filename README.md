 Simple Python Calculator

This is a beginner-friendly Python calculator that allows users to perform basic arithmetic operations:

✅ Addition (`+`)  
✅ Subtraction (`-`)  
✅ Multiplication (`*`)  
✅ Division (`/`), with error handling for division by zero.
